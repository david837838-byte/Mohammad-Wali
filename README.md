# نظام إدارة المستشفى

نظام متكامل لإدارة المستشفيات والمؤسسات الطبية بلغة Python باستخدام Flask.

## المميزات

- تسجيل دخول متعدد المستويات (مدير، طبيب، ممرض، مشاهد)
- إدارة المرضى والسجلات الطبية
- جدولة الزيارات والمتابعة
- تقارير وإحصائيات تفصيلية
- تصدير واستيراد البيانات
- تكامل مع Google Sheets (اختياري)
- واجهة مستخدم عربية متجاوبة

## متطلبات التشغيل

- Python 3.8 أو أعلى
- pip (مدير حزم Python)
